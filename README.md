# Challenge-Python-Data-Science

Este es un proyecto simple de Data Science para ayudar al Sr. Juan a decidir qué tienda de su cadena Alura Store debe vender para iniciar un nuevo emprendimiento. 
El objetivo es analizar los datos de ventas para obtener el rendimiento y promedio de reseñas de los clientes para cada una de las 4 tiendas.

Primer paso fue calcular el ingreso total de cada tienda.
Segundo paso fue calcular la cantidad de productos vendidos por categoría en cada tienda.
Tercer paso fue calcular las calificaciones promedio de los clientes para cada tienda.
Cuarte paso fue identificar qué productos fueron los más vendidos y los menos vendidos en cada tienda.
Quinto paso fue calcular el costo de envío promedio para cada tienda.
Por último, el sexto paso fue generar tres gráficos para visualizar los datos y realizar el análisis respectivo.

Para el proyecto se utilizaron las siguientes librerias:
- matplotlib.pyplot
- import pandas as pd

# Despúes de realizar todo el proceso se llega a la conclusión que la recomendación es vender la Tienda 4

Esto significa que no solo genera poco valor por venta, sino que sus clientes están altamente insatisfechos.
    2. Menor Volumen de Ingresos (Gráfico 1 - Barras): La Tienda 4 tiene el menor Ingreso Total de toda la cadena, lo que la hace la menos rentable en términos de volumen bruto.

Vender la Tienda 4 permite al Sr. Juan desinvertir en el activo con el peor desempeño financiero y la peor reputación de marca.
